# Description
This is a template for fyne Applications

# Requirements
You meet the Go Compiler. To compile from linux to Windows you need fyne-cross. To compile for Android you need the Android NDK. As well as make.

# Setup
Go to the Makefile and replace every instance of the word "template" with the name you want to give to your project. Run "make i" to setup the golang module. If you want to compile for Android, place the path to your Android NDK directory in the appropriate place. With make you can then compile for different targets.
